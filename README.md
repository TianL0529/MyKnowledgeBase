# MyKnowledgeBase

**Springboot-Notebook** 是一系列以 `springboot` 为基础开发框架，整合 `Redis` 、 `Rabbitmq` 、`ES` 、`MongoDB`、`sharding-jdbc`、`zookeeper` 等互联网主流技术，实现开发中常见功能点的综合实战性案例。

本着拿来即用的原则，助力于减少开发者在工作中的学习成本。
